LLaMA Chat Widget

Esse repositório hospeda o script widget.js usado pelo bookmarklet.

Para usar, crie um favorito com o seguinte código: 

javascript:(function(){
  var s=document.createElement('script');
  s.src='https://SEU_USUARIO.github.io/llama-widget/widget.js';
  document.body.appendChild(s);
})();

Depois clique no favorito em qualquer site para abrir/fechar o chat.
